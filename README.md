# vue-h5-pro

基于@vue/CLI3构建的移动端h5项目模板。

## 构建命令
### 初始化依赖配置
```
yarn install
```

### 开发环境 启动运行
```
yarn run serve
```

### 生产环境 打包构建
```
yarn run build

yarn build --report // 打包分析
```

### 自动化测试
```
yarn run test
```

### Lint检查并修复
```
yarn run lint
```
